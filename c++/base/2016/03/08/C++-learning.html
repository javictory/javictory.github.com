2、变量初始化

形式：<变量名>(表达式)

例如：int a(4); //与C语言中的int a=4;是等价的。

3、C++符号常量 const

C++有符号常量而C中是没有的

C++的格式： const <类型><符号常量>=值;

例如 ：const float PI=3.1415926; // C为 #define PI 3.1415926;

(1)与C相比，用const定义的符号常量具有数据类型，而且作用域可以是局部的。

(2)const 可以是参数指针不被改变

 例如： int sum(const int *a);    (3)调用时，不想改变引用变量a的值

 例如： void f(const int &a,int b){...}

4、自增判断

#include

using namespace std;

#define product(x) (x*x);

int main(){

int i=3,j,k;

j=product(i++);//此处的表达式为(i++)*(i++)，运行结束后，i==5；

k=product(++i);//此处的表达式为(++i)*(++i),在得到运算结果之前，i==7

cout<<"j="<<j<<"k="<<k<<endl;

system("pause");

return 0; }

在.cpp文件下无论是动态链接库的创建还是调用，都要声明一下extern “C” ；

6、在C中是没有string类型的，因此都是用字符数组进行 字符数组也没有办法直接进行字符串的赋值 只有在初始化的时候可以直接进行赋值
