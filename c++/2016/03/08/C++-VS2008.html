VC++ 6.0 编译器相关设置

8 MAR 2016 • 1 min read

1、结构体中所有成员的对齐参数N的最大值称为结构体的对齐参数。 

VC++6.0中n 默认是8个字节，可以修改这个设定的对齐参数；

(1)方法一：

在菜单“工程”的“设置”中的“C/C++”选项卡的“分类”中， “Code Generation ”的“Struct member alignment” 中设置：

1byte、2byte、4byte、8byte、16byte等几种，默认为8byte。

(2)方法二：

可以程序控制，采用指令： #pragma pack(xx)控制

 例如：
	#pragma   pack(1),1字节对齐，
	#pragma   pack(4)，4字节对齐，
	#pragma   pack(16)，16字节对齐

接下来我们将分不同的情况来详细讨论结构体成员的分布情况，顺便提醒一下， 常见类型的长度：

Int 4byte, Short 2byte, Char 1byte, Double 8byte, Long 4byte ； 

让我们先看下例： 

Struct  A 
{
	char   c; //1byte  
	double d; //8byte  
	short s;  //2byte  
	int   i;  //4byte  
};  

int main(int argc, char* argv[]) 
{ 
	A  strua;
	printf("%len:d\n",sizeof(A));     
	printf("%d,%d,%d,%d",&strua.c,&strua.d,&strua.s,&strua.i);  
	return 0;  
}

n设置为8byte时 结果：len:24,
