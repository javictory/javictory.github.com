在Windows上配置VIM

28 MAR 2016 • 1 min read

Vim是一款很牛逼的文本编辑工具，自己早已经习惯使用UE，但是久闻不如一见，试过才能评论好坏。 首先，由于Vim更多的人是用在了Linux上，关于Windows的安装配置相关的文章比较少，因此我决定在Windows7上尝试安装下Vim，并进行学习；

Vim的扩展性很强，在Github上有很多的Plugin可供你进行选择，安装和配置。但是，想必大家也了解，插件多，其实也未必是一件好事，因为随之而来的是管理问题。好在，大神们早已经为我们解决了这一苦恼的问题。那就是Vundle插件，一个专门用于管理插件的插件。 安装的过程在Vundle上的github有提供，详见：

https://github.com/VundleVim/Vundle.vim/blob/master/README_ZH_CN.md

里面有提供详细的安装过程，也许你之前已经有装过Gig了，但是为了以后少折腾建议还是先删除了，然后按照教程安装一遍。 大致的过程就是先安装（1）vim ；（2）github ； （3）Curl ；

我的安装目录是在 C：\Vim 下，git是安装在 C：\Git ;

首先是在C:\Vim\vimfiles下创建bundle文件夹，然后gitclone下Vundle；

$ git clone https://github.com/VundleVim/Vundle.vim.git

其次替换掉 C:\Vim_vimrc;暂时我的_vimrc是这个样子的

	set nocompatible              " 去除VI一致性,必须
	filetype off                  " 必须
	
	" 设置包括vundle和初始化相关的runtime path
	set rtp+=$vim/vimfiles/bundle/Vundle.vim
	call vundle#begin('$vim/vimfiles/bundle/')
	" 另一种选择, 指定一个vundle安装插件的路径
	"call vundle#begin('~/some/path/here')
	
	" 让vundle管理插件版本,必须
	Plugin 'VundleVim/Vundle.vim'
	
	" 以下插件安装.
	Plugin 'altercation/solarized'
	
	" 你的所有插件需要在下面这行之前
	call vundle#end()            " 必须
	filetype plugin indent on    " 必须 加载vim自带和插件相应的语法和文件类型相关脚本
	" 忽视插件改变缩进,可以使用以下替代:
	"filetype plugin on
	"
	" 简要帮助文档
	" :PluginList       - 列出所有已配置的插件
	" :PluginInstall    - 安装插件,追加 `!` 用以更新或使用 :PluginUpdate
	" :PluginSearch foo - 搜索 foo ; 追加 `!` 清除本地缓存
	" :PluginClean      - 清除未使用插件,需要确认; 追加 `!` 自动批准移除未使用插件
	"
	" 查阅 :h vundle 获取更多细节和wiki以及FAQ
	" 将你自己对非插件片段放在这行之后
	syntax on  
	set t_Co=256  
	let g:solarized_termcolors=256  
	set background=dark  
	colorscheme solarized

其中我指定了：

（1）rpt，vundle的运行库的位置；

	set rtp+=$vim/vimfiles/bundle/Vundle.vim

（2）Vim安装插件的位置；

	call vundle#begin('$vim/vimfiles/bundle/')

装好了一个基本的环境，我对Vim初始的配色方案很反感，并不是很喜欢。github上有solarized这个配色方案，看了下预览，感觉超赞。

然后我按照Vundle的插件管理方法，进行PluginInstall，安装完成后，重启gvim，发现还是原来的界面，还需要对其进行进一步设置；

solarized设置如下：	
	syntax on  
	set t_Co=256  
	let g:solarized_termcolors=256  
	set background=dark  
	colorscheme solarized

设置完成后，出现Error。后来发现还需要将 solarized.vim 拷贝到 C:\Vim\vim74\colors 下；再次打开，就能看到solarized风格的界面。
